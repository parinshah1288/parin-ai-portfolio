# Machine Learning vs Deep Learning

## Introduction

This artifact is a group presentation that explains the difference between Machine Learning (ML) and Deep Learning (DL), along with practical use cases for each.

## Objective

The objective of this presentation was to understand how machine learning and deep learning differ, and to identify where each approach is more suitable in real-world scenarios.

## Process

This was developed as a group presentation. We researched the key differences between machine learning and deep learning, focusing on how they work, the type of data they use, and their real-world applications.

We also explored examples where each approach performs better and explained the reasoning behind choosing one over the other.

## Tools Used

- Presentation slides  
- Research materials as listed in references in pdf.
- ChatGPT for minor help with structuring  

## Key Points from the Presentation

- Machine Learning works well with structured data and often requires manual feature engineering.  
- Deep Learning works well with unstructured data such as images, text, and video.  
- Machine Learning models are generally easier to understand and require less computational power.  
- Deep Learning models require large datasets and higher compute resources but can handle more complex problems.  
- Choosing the right approach depends on the type of data and the problem being solved.  

## Use Case: Machine Learning

In the presentation, we discussed how machine learning can be used for structured data problems such as airline ticket pricing.

In this case, the system uses inputs like number of available seats, time before departure, and historical pricing data. Based on these inputs, machine learning models can predict and adjust ticket prices dynamically.

This works well because the data is structured and the relationships can be modeled using traditional machine learning techniques.

## Use Case: Deep Learning

We also discussed deep learning in the context of security and surveillance systems.

In this case, the system processes images and video data to detect objects such as people or vehicles and identify activities in real time.

Deep learning models are effective here because they can automatically learn patterns from raw image data, which is difficult to handle using traditional machine learning.

## Value Proposition

This artifact provides a clear understanding of when to use machine learning versus deep learning. It highlights how selecting the right approach based on data type and problem complexity is important for building effective AI systems.
